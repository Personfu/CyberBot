## 🔨 Build & Deploy

```powershell
cd DreambotMasterAIO
.\build.ps1
```

Compiles 84 source files against `dreambot-client.jar`, packages `FLLCMasterAIO.jar`, and deploys to `%USERPROFILE%\DreamBot\Scripts\`. Refresh the DreamBot scripts list and launch **"FLLC Master AIO"**.
